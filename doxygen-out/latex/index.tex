C++11 tool for compile-\/time reflection utility through code generation.

Also includes some nice meta-\/programming utilities.

\doxysection*{A\+PI Documentation}

Doxygen output for the {\bfseries{latest version}} is available \href{https://briancairl.github.io/about/doxygen-out/html/index.html}{\texttt{ here}}

\doxysection*{Prerequisites}


\begin{DoxyItemize}
\item {\ttfamily apt-\/get install castxml python}
\item {\ttfamily pip install pygccxml}
\item {\ttfamily bazel} or {\ttfamily cmake}
\end{DoxyItemize}
